# Raccourcis clavier
## General
Ctrl-Shift-P : General command line
## Files
Ctrl-T : Open file from left pane
## Edition
- Ctrl-F : Find
- F3 : Find next

## Plugins

- Ctrl-Shift-H : Git-plus
- Ctrl-Shift-M : Markdownn preview
